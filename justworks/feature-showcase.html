Lightweight Health Score HS-1MVP ¬∑ Full Build

The same 5 signals from the status strip, now composed into a single score the agent reads in 1 second. Formula: Base = (Billing x 0.30) + (Payroll x 0.25) + (Support x 0.20) + (Satisfaction x 0.15) + (Activity x 0.10), then Final = Base x Tenure Factor. Tenure brackets: <30 days = 0.75, 30-90 days = 0.85, 90-180 days = 0.95, 180+ days = 1.00.

As a CSO agent during any interaction, I want a single composite health score that summarizes account health ‚Äî so I can instantly know whether this account needs special attention before diving into details.

As a CSO agent receiving a call about a failed payment, I want to see failure reason, account balance, and whether payroll is at risk ‚Äî so I can assess the crisis and guide the admin to resolution.
